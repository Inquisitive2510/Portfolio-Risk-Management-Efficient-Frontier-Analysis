# Portfolio Risk Management & Efficient Frontier Analysis

## Overview

This project implements Modern Portfolio Theory (MPT) techniques to evaluate portfolio construction and performance using:

- Mean-Variance Optimization
- Global Minimum Variance Portfolios
- Tangency Portfolios
- Fama-French Three-Factor Model
- Fama-French Five-Factor Model
- Efficient Frontier Analysis
- Practical Factor Proxies
- Mutual Fund Performance Evaluation

## Objectives

- Construct efficient frontiers for industry portfolios
- Compare sample estimates with Bayes-Stein estimators
- Analyze Fama-French factor portfolios
- Evaluate investable factor proxies
- Test portfolio performance in-sample and out-of-sample
- Assess mutual fund alpha and Sharpe ratios

## Technologies Used

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- SciPy
- Statsmodels
- Yahoo Finance Data
